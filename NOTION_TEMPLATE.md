**Step 3 - Paste URLs into pureinn-variables.md**

When you run `/pureinn` for the first time, the framework creates `pureinn-workspace/[slug]/pureinn-variables.md`.
Open that file and paste the URLs next to the matching keys.

Skills read this file automatically - you only fill it in once per project.

---

## FAQ

**Do I need to fill in all URLs?**
No. Leave rows blank and skills will ask you when they first need that item. You can fill them in incrementally.

**Can I use my own Notion setup instead of the template?**
Yes - paste the URLs of your own pages and databases. Skills match by URL, not by name. The template just gives you a pre-structured starting point.
