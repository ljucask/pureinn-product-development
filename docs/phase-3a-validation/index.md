# Phase 3a - Validation

Design and run experiments to validate (or invalidate) your riskiest assumptions. The phase ends with a hard Go/No-Go verdict.

**Duration:** Externally paced - tempo is set by the market, not by AI  
**Gate type:** Hard gate - only GO advances  
**Playbooks:** Greenfield

---

## When to enter this phase

Enter Phase 3a after Phase 2 (or when a Problem Validation Summary exists). Can start in parallel with late Phase 2.

**"Done elsewhere" rule:** if Design Thinking and hypothesis validation were done outside the framework, import results into `/pureinn` (Go/No-Go verdict, evidence per hypothesis, riskiest assumptions). Phase 3a is then complete - you don't re-run the experiments.

---

## What you need before entering

- Problem Validation Summary (Phase 2 exit artifact)
- Willingness to run real-world experiments (landing pages, smoke tests, pre-orders, concierge MVP)

AI accelerates experiment design and hypothesis structuring. The signal - whether people show up, convert, pre-order, confirm the pain - comes from the real world on its own schedule. This phase cannot be bypassed by AI.

## pm-hypotheses: two modes

`/pm-hypotheses` operates in two distinct modes:

**Plan mode** (after `/design-thinking`): Structures Validation Hypotheses into a formal register with ICP, assumption map, experiment plan, and success criteria defined *before* experiments run.

**Results mode** (after experiments): Records results against pre-defined criteria and issues the verdict:
- **GO** → advance to Phase 3b
- **PIVOT** → loop to targeted re-validation
- **STOP** → terminate the project

---

## What you exit with

- **Hypothesis Register** - ICP, assumption map, experiment plan, success criteria
- **Experiment results** - evidence collected from real-world validation
- **Go/No-Go verdict** - the Phase 3a exit decision

---

## Phase exit gate

**Gate type: Hard.** Only GO advances to Phase 3b. PIVOT loops to targeted re-validation with refined hypotheses. STOP ends the project.

There is no FORCE bypass. The verdict must be genuine GO based on experiment results.
